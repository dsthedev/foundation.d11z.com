Install Foundation

Installation

There are many ways to install Foundation, here are the main methods I use:

Install as CLI

  1. npm install --global foundation-cli
  2. foundation new --framework sites --template zurb/basic
  3. cd project/path && npm start

Install as Package

npm install --save foundation-sites

More ways to install can be found here.
